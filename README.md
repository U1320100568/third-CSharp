 實質型別(value)及參考型別(reference)  
 實質型別  
 即變數的記憶體(Stack)空間裡存放的為『內容(值)』  
 存真正資料，每個變數都擁有資料副本，如修改，不會影響其他變數  
 
 參考型別  
 即變數的記憶體空間(Stack)裡存放的為『Heap記憶體位址』  
 存位置指標，若多個變數參考到同份資料，修改後，其他變數會受影響
